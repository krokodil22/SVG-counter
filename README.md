# SVG Measure — v13

Сделано по запросу:
- Берём bbox как возвращает браузер.
- Вычитаем найденную толщину обводки из размеров:
  - `width = bbox.width - strokeWidth`
  - `height = bbox.height - strokeWidth`
  - позиция сдвигается внутрь на `strokeWidth/2` (чтобы расстояния считались от «итогового» прямоугольника)
- Расстояния считаются по этим скорректированным прямоугольникам.

Stroke W показывается в панели справа.
Все значения округляются до целого.
